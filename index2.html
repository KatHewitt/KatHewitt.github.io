Agent Based Modelling (ABM)

The following code describes the process of building an ABM which succeeds in representing individuals and capturing their interactions with each other and their environment. The final model builds up a series of agents in space which are able to communicate with each other and are able to query and change the environment based on the knowledge they have about the environment they are in.

The model simulates the movement of agents around the environment eating it, but first checking other agents are not already in the neighbourhood. The final model has been developed to include the ability for those agents who have consumed more food to move faster within the environment, simulating the idea that more food equals more energy and faster movement.

Floor Risk Modelling

The following code models the build up of water due to rainfall using DEM elevation data. The model produces an animation identifying those areas which are at risk from flooding.

By reading the imported DEM data the model calculates slope, which can then be used to determine the proportion of rain passed to multiple neighbours. An animation is produced in a new window which shows where water will end up - draining to the lowest elevations.

Drunkards Walk

The code builds a model which successfully simulates the random movement of drunk individuals leaving the pub, until they return themselves back to their home address.

The model starts by importing and reading some data about the environment. It then creates a series of agents, assigning each of them a house number and their start location at the pub. The model then simulates the random movement of the agents until they reach home. The code also creates a new environment in which the moves of each agent will be stored and a density map is created showing the steps of the agents.
